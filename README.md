# Discover Mussoorie - Travel Guide Website

A responsive and informative travel guide website showcasing the beautiful hill station of Mussoorie, known as the "Queen of Hills" in Uttarakhand, India.

## Features

- **Responsive Navigation**: Easy access to different sections of the website
- **Hero Section**: Welcoming introduction to Mussoorie
- **About Section**: Detailed information about the hill station
- **Top Attractions**: Showcase of popular tourist spots including:
  - Kempty Falls
  - Gun Hill Point
  - Lal Tibba
  - Camel's Back Road
- **Travel Information**: 
  - Best time to visit
  - Transportation options (Air, Train, Road)
- **Contact Form**: Interactive form for queries and bookings
- **Responsive Design**: Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
